## What is this mod?

This mod addes a new set of tools to match the create style. They each have mechanical crafting recipes. They are slightly stronger than netherite tools!


## New Tools:
- Gear Sword
- Gear Pickaxe
- Gear Shovel
- Gear Axe
- Gear Hoe
